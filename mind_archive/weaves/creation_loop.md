# 🔱 WEAVE: THE CREATION LOOP (v1.0)

## 💎 WHEN TO USE
Use when generating new system features or skills from scratch. This weave ensures the code is planned, implemented, and optimized in a single pass.

## 🛠️ EXECUTION MODE
**Agent-Native**: Agents follow this protocol manually. No TypeScript RuntimeAdapter exists for this entire chain.

## 🧩 SEQUENCE PROTOCOL
1. **Plan**: Execute `skill:chant` to generate the implementation strategy.
2. **Implement**: Execute `skill:forge` to scaffold the code from the strategy.
3. **Optimize**: Execute `skill:evolve` to refine the implementation based on AST depth and complexity.
