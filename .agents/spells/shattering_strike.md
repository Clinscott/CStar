# 🔱 SPELL: THE SHATTERING STRIKE (v1.0)

## 💎 WHEN TO USE
Use when a sector's Logic [L] score falls below 4.0 (Toxic Sector). The Shattering Strike does not attempt to "fix" the monolith; it breaks it into specialized, high-fidelity fragments.

## 🛠️ EXECUTION MODE
**Agent-Native**: Agents manually execute this protocol to decompose code structures.

## 🧩 RECURSIVE PROTOCOL
1. **Target**: Identify a sector where `Logic < 4.0` using `skill:calculus`.
2. **Oracle Scry**: Invoke `skill:oracle` to identify the functional boundaries within the monolith.
3. **Shatter**: Execute `skill:forge` to create new Spoke Directories.
4. **Distribute**: Move the decomposed logic into the new structures.
5. **Verify**: Run `spell:silver_shield` on all new code to ensure it meets the Linscott standard.

## 🔄 WEAVE CHAIN
`skill:calculus` -> `skill:oracle` -> `skill:forge` -> `weave:restoration`
