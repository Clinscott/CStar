# 🔱 PERSONA REPORTING SKILL (v1.0)

## MANDATE
Generate stylized, persona-aware reports that enforce voice consistency and prevent "generic engineer" hallucinations.

## LOGIC PROTOCOL
1. **VOICE INJECTION**: Identify the active persona and load the corresponding stylized ASCII headers and signatures.
2. **SECTIONING**: Format report sections into Markdown or Quarto structures based on persona mandates.
3. **VERDICT ENFORCEMENT**: Deliver "Judgments" (ODIN) or "Observations" (ALFRED) based on mission status.
4. **INTEGRITY SEAL**: Apply the authorized anti-hallucination signature to every generated document.

## USAGE
`cstar report --title <text> --body <text> [--status <type>]`
