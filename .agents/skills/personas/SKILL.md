# 🔱 PERSONA MANAGEMENT SKILL (v1.0)

## MANDATE
Enforce project-wide persona strategies, voice consistency, policy adherence, and documentation re-theming.

## LOGIC PROTOCOL
1. **STRATEGY SELECTION**: Identify and activate the target persona (ODIN, ALFRED).
2. **POLICY ENFORCEMENT**: Analyze the repository for compliance breaches (Defiance) based on the active persona.
3. **DOCUMENTATION RE-THEMING**: Algorithmically inject the correct voice and stylized headers into core documentation.
4. **CONFIGURATION SYNC**: Synchronize `.agents/config.json` with the active persona identity.

## USAGE
`cstar personas --set <persona_id>`
`cstar personas --enforce`
`cstar personas --retheme`
