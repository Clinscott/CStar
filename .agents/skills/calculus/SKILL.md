# 🔱 GUNGNIR CALCULUS SKILL (v1.0)

## When to Use
- Use when scoring code artifacts across logic, style, symmetry, and semantic intelligence.


## MANDATE
Perform multi-dimensional analysis of code artifacts to evaluate structural logic, aesthetic symmetry, and semantic intelligence.

## LOGIC PROTOCOL
1. **INTEL EXTRACTION**: Parse symbols and dependencies to map the sector's cognitive load.
2. **LOGIC VALIDATION**: Verify control flow and structural integrity against repository standards.
3. **STYLE ENFORCEMENT**: Score code for "Beauty" (Freya) and adherence to the Linscott Standard.
4. **GUNGNIR SCORING**: Synthesize all metrics into a single Gungnir Score for matrix promotion.

## USAGE
`cstar calculus --score --file <path>`
`cstar calculus --audit --file <path>`
