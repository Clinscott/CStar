# 🔱 SOVEREIGN SEARCH & STYLE SKILL (v1.0)

## When to Use
- Use when orchestrating neural search, cortex queries, and proactive skill discovery through the sovereign search engine.


## MANDATE
Orchestrate neural search, knowledge graph (Cortex) queries, and proactive skill discovery using the Master Sovereign Engine.

## LOGIC PROTOCOL
1. **NEURAL SEARCH**: Dispatch queries to the Sovereign Vector engine to find relevant skills or documentation.
2. **CORTEX QUERY**: Query the Knowledge Graph for deep relationship analysis between repository sectors.
3. **BENCHMARKING**: Compute diagnostic metrics for the vector engine (skill count, tokens, vectors).
4. **TRACE RECORDING**: Optionally record the neural path of every search mission.

## USAGE
`cstar style --query <text> [--json] [--record]`
`cstar style --cortex --query <text>`
`cstar style --benchmark`
