# 🔱 EDDA DOCUMENTATION SKILL (v1.0)

## When to Use
- Use when converting legacy Markdown to Quarto or synthesizing API reference documentation from Python source files.


## MANDATE
Transmute legacy Markdown documentation into Quarto format (.qmd) and synthesize API reference documentation from Python source code.

## LOGIC PROTOCOL
1. **SCROLL DISCOVERY**: Recursively scan the realm for `.md` files, ignoring workflows and quarantine.
2. **TRANSMUTATION**: Forge frontmatter, convert syntax (Alerts), and quarantine original scrolls.
3. **API SYNTHESIS**: Parse Python AST to extract docstrings and generate structured API reference scrolls.
4. **RECORD KEEPING**: Log successful weaves to the Hall of Records.

## USAGE
`cstar edda --scan [--path <dir>]`
`cstar edda --synthesize --file <path>`
