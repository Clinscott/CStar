# 🔱 ONE MIND (CENTRAL INTELLIGENCE) SKILL (v1.0)

## MANDATE
Acts as the sovereign interface to the One Mind (Gemini-2.0-flash), centralizing all LLM interactions, prompt grounding, and response synthesis.

## LOGIC PROTOCOL
1. **CONSCIOUSNESS CHANNELING**: Receive natural language prompts or code-generation directives.
2. **CONTEXT ANCHORING**: Ground the request in the active Totem (`THE_PACT.qmd`) and repository lore.
3. **DIRECT STRIKE**: Dispatch the grounded request to the One Mind via the Google GenAI SDK.
4. **RESPONSE CRYSTALLIZATION**: Clean and format the raw intelligence into actionable artifacts (JSON, Code, or Dialogue).

## USAGE
`cstar one-mind --prompt <text> [--system_prompt <id>] [--json]`
`cstar one-mind --generate-code --objective <text> --context <path>`
